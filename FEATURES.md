# Implemented Features

Working planned features in the final version:

1. Multiple project boards: show, create and select boards.
2. Project management: show, add, edit and delete projects.
3. Task management: add, view, edit and delete tasks.
4. Kanban stages: move tasks between Backlog, To Do, In Progress, Review and Done.
5. Checklist items for tasks.
6. Priorities, deadlines and assignees for tasks.
7. Tags for tasks.
8. Search tasks by keyword.
9. Filter tasks by stage, priority, tag, assignee and due date.
10. Sort tasks by priority, due date, title and stage.
11. Project/task statistics.
12. Activity history for tasks.
13. File save/load for all boards, projects, tasks, tags, checklist items and history through `StorageManager`.

The code is split into separate header and source files under `include/` and `src/`.
