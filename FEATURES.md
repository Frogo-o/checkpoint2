# Implemented Features

This checkpoint version intentionally implements a smaller working scope.

Working planned features:

1. Project board with seeded data.
2. Project management: show and add projects.
3. Task management: add and view tasks.
4. Kanban stages: move tasks between stages.
5. Checklist items for tasks.
6. File save/load through `StorageManager`.

Partially represented in the class model, but not exposed as full menu features yet:

- Tags.
- Activity history.
- Search/filter/sort/statistics through a future `TaskController`.
