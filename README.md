# Task and Project Manager

Terminal C++ OOP application for project boards, projects and tasks.

## Build

```powershell
g++ -std=c++17 -Wall -Wextra -pedantic src/main.cpp -o task_manager.exe
```

or:

```powershell
.\build.ps1
```

## Run

```powershell
.\task_manager.exe
```
